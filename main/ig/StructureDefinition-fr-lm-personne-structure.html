Logical Model: Modèle logique métier - FR LM Personne et/ou Structure

Official URL: https://interop.esante.gouv.fr/ig/document/core/StructureDefinition/fr-lm-personne-structure Version: 0.1.0
Draft as of 2025-10-21 Computable Name: FRLMPersonneStructure

Une personne (professionnel ou patient ou autre) et/ou une structure (pour les professionnels).

Usages:

  • Use this Logical Model: Modèle logique métier - FR LM Destinataire prévu, Modèle logique métier - FR LM Évènement, Modèle logique métier - FR LM Informateur, Modèle logique métier - FR LM Opérateur de saisie... Show 4 more, Modèle logique métier - FR LM Participant, Modèle logique métier - FR LM Prise en charge, Modèle logique métier - FR LM Responsable and Modèle logique métier - FR LM Validateur
