Logical Model: serviceEvent

Official URL: https://interop.esante.gouv.fr/ig/document/core/StructureDefinition/fr-core-service-event Version: 0.1.0
Draft as of 2025-02-05 Computable Name: FrServiceEvent

ServiceEvent représente un évènement (acte, traitement, diagnostic, etc…) décrit dans le document. L’occurrence de documentationOf/serviceEvent contenant les données de l’évènement documenté principal doit inclure un élément effectiveTime et un élément performer renseignés, sans recours à l’attribut nullFlavor.

Usage:

  • Use this Logical Model: documentationOf
