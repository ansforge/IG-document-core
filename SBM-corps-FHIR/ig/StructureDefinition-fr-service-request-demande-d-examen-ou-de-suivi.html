Resource Profile: ServiceRequest - Fr Demande d'examen ou de Suivi

Official URL: https://interop.esante.gouv.fr/ig/document/core/StructureDefinition/fr-service-request-demande-d-examen-ou-de-suivi Version: 0.1.0
Draft as of 2025-09-04 Computable Name: FrSeviceRequestDemandeDexamenOuDeSuivi

Ce profil permet de porter des demandes d’examens (analyses biologiques, évaluations, étude d’imagerie, etc…) ou de suivis particuliers à programmer dans le cadre d’un plan de soins.

Usages:

  • This Profile is not used by any profiles in this Implementation Guide

You can also check for usages in the FHIR IG Statistics
