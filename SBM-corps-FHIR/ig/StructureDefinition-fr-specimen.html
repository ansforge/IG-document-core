Resource Profile: Specimen - Fr Prélèvement

Official URL: https://interop.esante.gouv.fr/ig/document/core/StructureDefinition/fr-specimen Version: 0.1.0
Draft as of 2025-09-04 Computable Name: FrSpecimen

FrSpecimen est un profil utilisé pour décrire le prélèvement et l’échantillon biologique (le matériel).

Usages:

  • Refer to this Profile: Observation - Fr Acte biologie prescrit, Observation - Fr Batterie examen biologie médicale, Media - Fr Media image illustrative, Observation - Fr Isolat microbiologique... Show 2 more, Observation - Fr Resultats examens de biologie élément clinique pertinent and Specimen - Fr Prélèvement
