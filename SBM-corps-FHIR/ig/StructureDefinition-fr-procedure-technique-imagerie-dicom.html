Resource Profile: Procedure - Fr Technique imagerie DICOM

Official URL: https://interop.esante.gouv.fr/ig/document/core/StructureDefinition/fr-procedure-technique-imagerie-dicom Version: 0.1.0
Draft as of 2025-09-04 Computable Name: FrProcedureTechniqueImagerieDICOM

FrProcedureTechniqueImagerieDICOM permet d’enregistrer les différents paramètres de l’acquisition d’image : acte d’imagerie, modalité d’acquisition, localisation anatomique / latéralité / topographie, d’autres paramètres de l’acte

Usages:

  • This Profile is not used by any profiles in this Implementation Guide
