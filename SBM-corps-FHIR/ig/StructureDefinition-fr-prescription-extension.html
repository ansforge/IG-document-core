Extension: Extension - Fr Prescription

Official URL: https://interop.esante.gouv.fr/ig/document/core/StructureDefinition/fr-prescription-extension Version: 0.1.0
Active as of 2025-09-04 Computable Name: FrPrescriptionExtension

Extension permettant d’indiquer les instructions de prescription associées à l’administration du produit de santé.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: Resource

Usage info

Usages:

  • Use this Extension: MedicationAdministration - Fr Traitement, MedicationStatement - Fr Traitement, ImmunizationRecommendation - Fr Vaccin Recommande Document and Immunization - Fr Vaccination Document
