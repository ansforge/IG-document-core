Resource Profile: MedicationAdministration - Fr Administration Produits De Santé DICOM

Official URL: https://interop.esante.gouv.fr/ig/document/core/StructureDefinition/Fr-administration-produits-de-sante-DICOM Version: 0.1.0
Draft as of 2025-08-25 Computable Name: FrAdministrationProduitsDeSanteDICOM

FrAdministrationProduitsDeSanteDICOM permet d’enregistrer l’administration de produits (hors radiopharmaceutiques) : produit administré, dose administrée, voie d’administration, numéro de lot, autres informations liées à l’administration d’un produit

Usages:

  • This Profile is not used by any profiles in this Implementation Guide

You can also check for usages in the FHIR IG Statistics
