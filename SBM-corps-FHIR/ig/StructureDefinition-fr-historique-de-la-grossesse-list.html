Resource Profile: List - Fr Historique de la grossesse

Official URL: https://interop.esante.gouv.fr/ig/document/core/StructureDefinition/fr-historique-de-la-grossesse-list Version: 0.1.0
Draft as of 2025-09-03 Computable Name: FrHistoriqueGrossesseList

Liste contenant soit un événement de naissance, soit une observation sur la grossesse, mais pas les deux.

Usages:

  • This Profile is not used by any profiles in this Implementation Guide
